webdav-js v0.2

A browswer-based WebDAV client, written in JavaScript.

Information

This simple WebDAV browsing/uploading client is fairly light and a useful addition to your bookmarks. It supports previewing of moany common filetypes (syntax highlighting for code, previews for images/videos/fonts), drag and drop file uplaods and history state (for back button navigation).

Whilst this started out as a very simple bookmarklet with some basic styling (and whilst it's still that!), I'd like to improve it somewhat, perhaps using jsTree to handle a copy/move dialog and test it more thoroughly, but I use it personally fairly regularly for minor file changes.
